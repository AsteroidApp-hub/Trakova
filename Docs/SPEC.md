| 項目 | 内容 |

# v1.0 リリース対象機能（実装済み）

## 1. 録音

| 機能 | 仕様 |
|------|------|
| マルチトラック録音 | mono / stereo・入力チャンネル選択 |
| レーン録音 | テイクごとに Take レーンへ自動バックアップ |
| ループ録音 | LOOP 範囲設定 + Rec で自動 |
| パンチイン録音 | 再生中に R で開始 → 既存クリップ上にオーバーレイ録音 |
| 遡及録音 | 再生中バックグラウンドキャプチャ → Cmd+Shift+R で確定 |
| プリロール | 0 / 1 / 2 / 3 秒 |
| カウントイン | 0 / 1 / 2 / 4 小節 |
| メトロノーム | 専用 CLICK トラック・Beep / Stick / Cowbell / Wood の 4 種類・1 拍目アクセント・Half / Normal / Double レート |
| タップテンポ | ツールバーの TAP ボタン |
| 入力レベルメーター | 録音前に Rec アーム or Input Monitor で確認 |
| テイク比較 | Take レーン (Shift+↑ で Lane 0 へ採用) |
| 自動リサンプリング | 異なる SR の WAV を r8brain で変換 |
| パンチイン境界クロスフェード | 自動 30ms フェード (クリック防止) |

> ⚠ 同時録音は **1 トラックのみ** (最初に Rec アームしたトラックを使用)

## 2. 編集

| 機能 | 仕様 |
|------|------|
| クリップ操作 | 移動・リサイズ・分割 (Option/Alt+Click)・結合 (Consolidate) |
| フェード | Linear / Logarithmic / EqualPower / SCurve |
| クロスフェード | F = 単独 fade / X = 隣接 xfade・両端ハンドルでサイズ調整 |
| クリップゲイン | フェーダー + ブレークポイントエンベロープ |
| 無音区間カット | Strip Silence |
| BPM 検出 | 波形右クリック → 「テンポを検出」 |
| ピッチシフト | -6〜+6 半音・キャッシュ付き (Audio フォルダ内に保存) |
| Undo / Redo | JUCE UndoManager (LaneSnapshotAction で複雑操作も網羅) |
| スナップ | Off / 1/1 / 1/2 / 1/4 / 1/8 / 1/16 / 1/32 / 1/4T / 1/8T / 1/16T |
| クリップ名称変更 | 各クリップに任意の名前 |
| クリップカラーリング | 8 色から選択 |
| ナッジ | 矢印キーで微小移動 |
| スマートツール | 上半分=範囲選択 / 下半分=ハンド・自動切替 |
| 自動レーン退避 | ドラッグ重ね時、下側を Take レーンへ自動移動 |

## 3. ミキサー（トラック上）

| 機能 | 仕様 |
|------|------|
| フェーダー | 横型スライダー (Vol) |
| Pan | スライダー (中央 = 0) |
| Rev | 簡易リバーブセンド (0..1) |
| VU / Peak メーター | IEC 268-17 準拠 300ms 積分・SR/blockSize 非依存・ピークホールド付き |
| ミュート / ソロ / Rec アーム / Input Monitor | トラックヘッダの M/S/R/I |
| INS スロット | 固定 4 枠 (VST3 / AU)・自己管理エディタウィンドウ |
| INS D&D | スロット間移動・Option+Drag コピー (別トラック間も可) |
| INS バイパス | Cmd+Click で個別バイパス |
| PDC | プラグインディレイ補正 (TrackDelay 構造体) |
| トラック複製 | 右クリックメニュー「トラックを複製」(クリップ・設定・INS をクローン) |

# 不採用

検討の結果、Utawave としては実装しないと決めた項目。

| 項目 | 理由 |
|------|------|
| **VST2 対応** | JUCE 8 で SDK サポートが廃止された |
| **64-float bit 書き出し** | 実用上の需要なし。32-float で十分 |
| **内蔵ファイルブラウザ** | OS のファイラ + D&D で十分 |
| **Collect All** | インポート時点で常に `<projectFolder>/Audio/` へコピーするため既に集約済み |
| **MIDI インスペクターウィンドウ** | ピアノロールエディタが完全な編集機能を提供するため不要 |
