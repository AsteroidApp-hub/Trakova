01はじめに・画面の見方

Utawave は、歌ってみた・カバー投稿のための「録音」に特化した DAW です。むずかしい設定をなるべく省き、インスト音源を入れて、録音して、書き出すという流れを最短で行えるように設計しています。

画面の構成

  • 上部（ツールバー）：再生 / 停止 / 録音ボタン、再生位置の時間表示、テンポ（BPM）、メトロノームやループの切り替え。
  • 左（TRACKS パネル）：トラックの一覧。各トラックの音量（Vol）・左右（Pan）・残響（Rev）・M ミュート・S ソロ・R 録音アーム・I 入力モニターなど。
  • 中央（タイムライン）：録音した声や取り込んだ音源が「波形」として横に並びます。ここで切ったり動かしたりします。
  • 右（MASTER パネル）：曲全体の最終音量とメーター。折りたたみできます。
ヒント：多くの操作はボタンとキーボードの両方から行えます。まずはボタンで慣れて、慣れてきたらショートカットを使うと作業が速くなります。

02プロジェクトの基本

作業内容は「プロジェクト」という単位で保存します（ファイルの拡張子は .utawave）。曲ごとに 1 つのプロジェクトを作るイメージです。

新規・開く・保存

  • 新規：「ファイル」→「新規プロジェクト...」
  • 開く：「ファイル」→「開く...」（Ctrl+O）。起動画面の「最近使ったプロジェクト」からも開けます。
  • 保存：Ctrl+S。別名で保存は Ctrl+Shift+S。

自動保存とバックアップ

一定時間ごとに自動で保存され、さらに日時つきの世代バックアップが残ります。万一アプリが終了しても、次に同じプロジェクトを開いたときに復旧するかたずねられます。

音源ファイルはプロジェクトの中に入ります：取り込んだ音や録音は、プロジェクトフォルダ内の Audio フォルダにコピーされます。プロジェクトを他者へ渡すときはフォルダごと渡してください。

03インスト音源を取り込む

まずは伴奏（オフボーカル / インスト音源）を読み込みます。

  1. 「ファイル」→「オーディオを読み込む...」（Ctrl+I）を選ぶ。または音源ファイルをタイムラインにドラッグ＆ドロップ。
  2. 取り込んだ音源は自動的に曲の先頭（0 秒）へ配置されます。

対応形式：WAV / AIFF / MP3 / FLAC / Ogg など一般的な形式に対応しています（macOS では m4a / aac なども可。対応形式は OS によって一部異なります）。サンプルレートが異なるファイルは自動で変換して読み込みます。

音量も自動でそろえます：取り込み時に音量（ラウドネス）を目安の大きさへ自動調整します（設定でオフにできます）。

04録音の準備をする

  1. トラックを追加：TRACKS パネル左下の + ボタン（または下部の「+ Add Track」）から、モノラル／ステレオを選びます。歌の録音は通常モノラルでかまいません。
  2. 入力を選ぶ：トラックの「In:」から、使うマイク（入力チャンネル）を選びます。
  3. 録音アーム：録音したいトラックの R ボタンを点灯させます。これで「このトラックに録る」状態になります。
  4. 声を確認：I（入力モニター）を点灯すると、自分の声が返ってきて、メーターも動きます。声を出してメーターが振れることを確認しましょう。
複数同時録音：R を複数のトラックで点灯させ、それぞれ別の入力を選べば、同時に複数トラックへ録音できます。
音が出ない・入力が出てこないときは：「ファイル」→「環境設定」のオーディオ設定で、使用するオーディオ機器（オーディオインターフェース）が選ばれているか確認してください。

05テンポとメトロノーム

  • テンポ（BPM）：ツールバーの BPM 表示をダブルクリックすると数値を直接入力できます。
  • タップで決める：TAP ボタンを曲に合わせて叩くと、テンポを推定できます。
  • 自動で当てる：取り込んだ音源を右クリック →「テンポを検出」で、伴奏から BPM を自動推定できます。
  • メトロノーム：CLICK ボタンでクリック音のオン / オフ。クリック専用トラックで音量なども調整できます。
  • カウントイン：録音前に何小節カウントを鳴らすか設定できます（環境設定）。

06録音する

  1. 録りたいトラックの R が点灯していることを確認。
  2. 再生位置を録音開始位置に合わせる（ルーラーをクリック、または N / B で 1 小節ずつ移動）。
  3. R キー（またはツールバーの録音ボタン）で録音開始。カウントインの設定があれば鳴ってから始まります。
  4. もう一度 R、または Space で停止。

便利な録音のしかた

  • パンチイン：再生しながら録り直したい箇所で R を押すと、その部分だけ上書き録音できます（つなぎ目は自動でなめらかに処理）。
  • ループ録音：L でループ範囲を繰り返し再生しながら、何テイクも録れます。
  • さかのぼって確定：再生中の歌はうしろで常にキャプチャされています。「今のがよかった」と思ったら Ctrl+Shift+R でさかのぼって録音を確定できます。

07テイクの管理

同じ場所に録り直すと、前のテイクは消えずにテイクレーンへ自動的に退避されます。聴き比べて、良いテイクを選べます。

  • トラックの TList ボタンでテイクレーンの表示 / 非表示を切り替えます。
  • 各テイクの S（ソロ）でそのテイクだけを聴けます。
  • 採用したいテイクの ↑ ボタン（または Shift+↑）で、そのテイクをメインに昇格できます。

08波形を編集する

タイムライン上の音のかたまりを「波形」と呼びます。マウスで直感的に編集できます。

フェードとクロスフェード

  • フェードイン／アウト：範囲を選んで F。出だしや終わりをなめらかにします。
  • クロスフェード：2 つの波形の重なり／接した部分を選んで X。つなぎ目のプチッというノイズを防げます（右クリック →「クロスフェードを描く」でも可）。

音量の調整

  • 波形ごとの音量：波形のフェーダーで上げ下げ。波形の中で部分的に音量を変えたいときは、音量ラインに点を打って（ブレークポイント）細かく調整できます。
  • 無音をカット：右クリック →「無音をカット」で、息継ぎなどの静かな部分を自動的に取り除けます。
表示の拡大・縮小：Ctrl＋マウスホイールで横方向にズーム、Shift+F で曲全体が画面に収まります。

09キー変更・テンポ検出

  • キー（音程）を変える：波形を右クリック →「キーを変更...」で -6〜+6 半音から選べます。高品質な処理で音質を保ったまま移調します。結果は新しいファイルとして保存され、同じ設定なら次回はすぐ反映されます。「0（元の音程）」で元に戻ります。
  • テンポを調べる：波形を右クリック →「テンポを検出」で、伴奏の BPM を自動推定します。

10エフェクト（FX）

リバーブやコンプレッサーなどのプラグインを、トラックや全体（マスター）に追加できます。

FX スロットを開く

TRACKS パネル左下の FX ボタンを押すと、各トラックの右側にエフェクト用のスロット（INS）が表示されます。もう一度押すと隠れます。

  • 空きスロットをクリック → 使いたいプラグインを選んで追加。
  • 追加したプラグインのチップをクリックすると設定画面が開きます。
  • Ctrl＋クリックで一時的にオフ（バイパス）。ドラッグで並べ替えや別トラックへ移動（Alt＋ドラッグでコピー）。
プラグインが出てこないときは：「ファイル」→「プラグインを管理...」を開き、スキャンを実行してください。インストール済みのプラグインが一覧に表示されます。

リバーブ（残響）を加える

各トラックの Rev スライダーで、そのトラックに残響（リバーブ）を加えられます。録音中の「返し（モニター）」にのみ適用することもでき、その場合でも録音ファイルには書き込まれません。

11MIDI を読み込む・打ち込む

MIDI データ（.mid ファイル）を読み込んで、ガイドメロディやハモリ、伴奏として利用できます。手で打ち込むこともできます。

MIDI ファイルを読み込む

  1. 「ファイル」→「MIDI を読み込む...」を選び、.mid ファイルを開きます（タイムラインへのドラッグ＆ドロップでも可）。
  2. 「MIDI を読み込む」ダイアログで、読み込みたいトラックにチェックを入れます（各トラックのチャンネルやノート数が表示されます）。
  3. 配置を「先頭に配置」または「現在の再生位置に配置」から選びます。必要に応じて「テンポ・拍子も取り込む」「マーカーも取り込む」にチェックします。
  4. 「読み込み」を押すと、選んだトラックが MIDI トラックとして追加されます。
音を鳴らすには：読み込んだ MIDI トラックは内蔵の音色で再生されます。お好みの音源プラグインを挿せば、その音色で鳴らせます（エフェクト（FX）と同じ要領で追加します）。

手で打ち込む（ハモリ・ガイド作成）

  • トラック追加メニューから「MIDI トラックを追加」を選びます。
  • タイムラインをダブルクリックすると打ち込み用の区間ができ、その区間をダブルクリックするとピアノロール（鍵盤の編集画面）が開きます。
  • ピアノロールでは、空き部分のダブルクリックで音符を追加、ドラッグで移動／長さ変更、Delete で削除。↑ / ↓ で半音上下。

12書き出す（WAV / MP3）

完成したら音声ファイルに書き出します。

  1. 「ファイル」→「書き出し...」（Ctrl+E）を開く。
  2. 形式（WAV / AIFF / MP3）と範囲（曲全体／選択範囲／小節範囲）を選ぶ。
  3. ミックス全体を 1 ファイルにするか、トラックごと（ステム）に分けるかを選んで書き出します。
音割れ防止：書き出し時、音が大きすぎる場合は自動で少し下げて割れを防ぎます（設定でオフにできます）。

14困ったときは

「音声ファイルが見つかりません」と表示される

プロジェクトが参照している音源が移動／削除された可能性があります。元のファイルをプロジェクト内の Audio フォルダに戻すと自動的に復活します。

前回の作業が消えた／アプリが終了した

保存済みプロジェクトを開き直すと、より新しいバックアップがある場合に復旧をたずねられます。「復旧」を選ぶと直前の状態に戻せます。

マイクの声が録れない

  • OS のマイク使用許可で Utawave が許可されているか確認。
  • 環境設定のオーディオ設定で、正しい入力機器とチャンネルが選ばれているか確認。
  • トラックの R（録音アーム）が点灯しているか確認。

自分の声が遅れて聞こえる（モニターの遅延）

Windows でオーディオインターフェイスをお使いの場合は、オーディオ設定でドライバの種類に ASIO（インターフェイス付属のドライバ）を選ぶと大きく改善します。あわせてバッファサイズを小さくすると遅れが減ります（小さくしすぎて音が途切れる場合は一段階戻してください）。

録音した歌がオケより遅れて聞こえる

Utawave は録音時に機器の遅延（レイテンシ）を自動で補正します。それでもずれる場合は、環境設定の「録音フロー」にある追加の録音補正 (ms) を少しずつ増やして調整してください（+で手前に移動します）。

プラグインが一覧に出ない

「ファイル」→「プラグインを管理...」で「全て再スキャン」を実行してください。インストール直後のプラグインも認識されます。動作に問題のあるプラグインはスキャン時に自動で除外されます（アプリは落ちません）。
