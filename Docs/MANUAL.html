1. インストール

対応 OS

  • macOS 12 (Monterey) 以降 ※ Apple Silicon (M1 以降) 専用

インストール手順

  1. 配布された Trakova.app をダウンロード (zip の場合は展開)
  2. Trakova.app を アプリケーションフォルダ へドラッグ
  3. 初回起動: Finder で 右クリック → 開く を選択。「開発元を確認できません」と出ても 開く をクリック
    (Apple のコード署名/公証を取得していないため、通常のダブルクリックでは Gatekeeper にブロックされます)
注意: 2 回目以降は通常通りダブルクリックで起動できます。
なお、Apple ID で署名されていない開発版アプリのため、初回のみ macOS 側で許可が必要です。

2. 初回起動

起動すると スタート画面 が表示されます。

  • 新規プロジェクト ― サンプリングレート / ビット深度 (32-bit がおすすめ) を指定して作成
  • プロジェクトを開く ― 既存の .trakova ファイルを開く
  • 最近使ったプロジェクト ― リストから素早く再開

3. 基本概念

用語説明
プロジェクト1 曲分の作業データ。.trakova ファイル + 同じフォルダの Audio / Cache / Backup / Export サブフォルダで構成。
トラック音源 1 つ分の入れ物。インスト用・ボーカル用などを別々のトラックに分けて配置。
クリップトラック上に配置される音声/MIDIブロック。録音やインポートで生成される。
テイク (Take)1 つのトラックは複数のテイクを持てる (テイクリスト)。Take 0 がメイン (再生対象)。Take 1 以降は録り溜めたテイクの控え。
マスター画面右側のパネル。全トラックがミックスされた最終出力に対するエフェクトとフェーダー。

4. 新規プロジェクトを作成

  1. 起動画面で 新規プロジェクト をクリック
  2. 保存先フォルダとプロジェクト名を指定
  3. サンプリングレート (44.1k / 48k / 96k 等) を選択
  4. ビット深度 (16 / 24 / 32-bit) を選択
  5. 作成すると MyProject.trakova と Audio 等のフォルダが生成される
プロジェクトを作成すると、オーディオデバイスのサンプリングレートも自動的にプロジェクト設定へ追従します。

5. インスト (伴奏) を読み込む

方法 A: ドラッグ & ドロップ

Finder から音声ファイル (WAV / AIFF / MP3 / M4A / FLAC) をタイムラインへ直接ドロップ。トラックが自動生成され、適切な場所にクリップが配置されます。

方法 B: メニューから取り込み

Cmd + I または File メニュー → オーディオファイルを取り込む

サンプリングレートが違う場合: 自動的にプロジェクトのサンプリングレートにリサンプリングされます。

ラウドネス自動調整 (オプション)

環境設定 → インポート時にラウドネスを自動調整 を ON にすると、インポート時に LUFS を測定して自動的にゲインを補正します (ターゲット: 既定 -24 LUFS、ATSC A/85 基準)。

6. ボーカルを録音する

準備

  1. ツールバー右上 AUDIO ボタン → 録音デバイス (オーディオインターフェース) を選択
  2. 新規ボーカル用トラックを追加 (左パネル + Add Track → モノラル/ステレオ選択)
  3. トラックの In: プルダウンから録音入力チャンネルを選択
  4. トラックの R (録音) ボタンをクリック → 赤く点灯 (このトラックが録音対象になる)
  5. I (Input Monitor) ボタンで自分の声を返したい場合は ON にする

録音開始

操作動作
キーボード R (停止中)録音開始 (カウントイン / プレロールの設定があればその後に開始)
キーボード R (再生中)パンチイン録音 — その場で録音開始 (既存音は流れたまま)
キーボード Space再生 / 停止

カウントイン / プレロール

  • C-IN (カウントイン): 1/2/4 小節 ― 録音開始位置から指定した小節数だけ手前に戻り、メトロノームのカウントを鳴らしてから録音を開始します
  • PRE (プレロール): 1/2/3 秒 ― 録音開始位置の指定秒数だけ手前からインストを再生し、その後に録音を開始します (歌い出しを掴みやすい)

どちらもツールバー (BPM の右側) で切り替えます。

遡及録音 (Retrospective)

環境設定で 再生中バックグラウンド録音 が ON の場合、再生中は常に裏で録音されています。

ループ録音

  1. ルーラーをドラッグしてループ範囲を指定 (オレンジの帯が出る)
  2. ツールバー LOOP ボタンを ON
  3. R で録音 → ループするたびに別のテイク (Take 1, Take 2 …) に保存される

7. テイクの管理 / 差し替え

テイクリストの表示

トラックヘッダーの TList ボタンを ON にすると、Take 1 以降のテイクが下に展開されます。

気に入ったテイクを採用する

  1. テイクリスト上 (Take 1 以降) のクリップを 右クリック
  2. メニュー最上段の 「このテイクを使う」 を選択
  3. そのクリップが Take 0 (メイン) に差し替えられる

ショートカット: Shift + ↑

9. ミックスとプラグイン

トラックフェーダー / パン

トラックヘッダーの Vol / Pan スライダーで調整。ダブルクリックで 0 dB / センターに戻る。

M / S / R / I ボタン

各ボタンは クリックで切り替えます。キーボードで操作する場合は Shift + キー を使います (選択中のトラックが対象)。

ボタンキーボード機能
MShift + MMute (オレンジ点灯時に無音)
SShift + SSolo (青点灯時、このトラックだけ再生)
RShift + R録音対象にする (赤点灯時)
IShift + IInput Monitor (緑点灯時、入力をスピーカ/ヘッドホンへ返す)

メーター

  • Peak — 即時ピーク値。緑 (〜-6dB) → 黄 (-6〜0dB) → 赤は 0 dBFS を超えた "ピークオーバー" の部分のみ
  • VU — 平均的なラウドネス感 (300ms 積分)
  • VU の 0 VU 基準ライン はオレンジの縦線。環境設定で -14/-18/-20/-24 dBFS から選択可
  • マスターの CLIP! 表示は 0 dBFS を超えると点灯し、RESET ボタンを押すまで保持されます (一瞬の超過も見逃しません)

プラグイン

  1. 初回: メニュー → プラグイン管理 でスキャン
  2. トラックヘッダーで INSERTS 列を右クリック → INS スロットを表示
  3. 空きスロットの + をクリックしてプラグインを選択
  4. スロットをクリックしてエディタを開く
  5. Cmd + クリックでバイパス切替
  6. ドラッグで別トラックへ移動、Option + ドラッグでコピー

マスターチェーン / ディザー

右側 MASTER パネルにマスターバス用のプラグインを挿せます。一番下の DITHER スロットは 24bit 未満で書き出すときの高品質ディザ用。

9.5 MIDI トラック (ハモリ / ガイドメロディ)

ハモリやガイドメロディを打ち込みたいときは MIDI トラックを使います。

  1. + Add Track メニュー → MIDI トラックを追加 (空のトラックが作られます)
  2. タイムラインの空きエリアで ダブルクリック = 1 小節の MIDI クリップ作成 / Option + ドラッグ = ドラッグした長さで作成
  3. クリップの 左右端ドラッグでリサイズ、タイトル部ドラッグで移動 (グリッドにスナップ)
  4. クリップを ダブルクリックするとピアノロールが開きます
  5. ピアノロールでノートを打ち込み (操作はショートカット表を参照)
音源は 内蔵シンセ (波形右クリックで ON/OFF) か、INS スロットに挿した 音源プラグイン を使えます。 SMF (.mid) ファイルの読み込みにも対応しています (メニュー → MIDI を読み込む)。

10. 書き出し (WAV / MP3)

  1. Cmd + E または メニュー → 書き出し
  2. 書き出し範囲: プロジェクト全体 / 選択範囲 / マーカー間 から選択
  3. 形式: WAV / AIFF / MP3 (MP3 エンコーダ内蔵)
  4. ビット深度・サンプリングレートを指定
  5. Pre-fader (フェーダー前) / Post-fader (通常) のどちらでレンダリングするか選択
  6. 書き出し をクリック
プロジェクトフォルダの Export/ フォルダがデフォルトの保存先になります。

13. トラブルシューティング

Q: 「開発元を確認できません」と出て起動できない

A: Finder で 右クリック → 開く を選択し、確認ダイアログで 開く をクリックしてください。次回からは通常のダブルクリックで OK です。

Q: 音が鳴らない / 録音できない

A: 右上の AUDIO ボタンからオーディオデバイスを確認してください。USB マイク等は OS の「サウンド入力」を Trakova 側で選び直す必要があります。

Q: プラグインが見つからない

A: 初回はメニュー → プラグイン管理 でスキャンしてください。スキャン後、トラックの INS スロットから挿せます。

Q: 「"Trakova" は壊れているため開けません」と出る (Apple Silicon)

A: ダウンロードした zip に付く検疫属性が原因です。ターミナルで次を 1 回実行してください:
xattr -cr /Applications/Trakova.app
その後は通常どおり起動できます (アプリを「アプリケーション」へ入れてから実行)。

Q: 大きなプロジェクトの波形読み込みが遅い

A: 初回は波形を生成しますが、2 回目以降はプロジェクト内の Cache/ に保存された波形を使うため高速に開きます。ステータスバーに読み込み進捗が表示されます。

Q: CPU 使用率が高い

A: 右下 ステータスバーの CPU% を確認。多数のプラグインを使う場合は バッファサイズを増やしてください (AUDIO 設定)。

Q: 自動保存はどこに?

A: プロジェクトフォルダ内の Backup/ サブフォルダに <name>.autosave.trakova として保存されています。クラッシュ後の起動時に復旧候補があれば自動でダイアログが出ます。
