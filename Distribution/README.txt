Utawave — 歌い手向け DAW （テスト版 v0.1.0）

このたびはテストにご協力いただきありがとうございます。
録音特化の無料 DAW です。気づいた点をぜひ教えてください。

===================================
■ インストール
===================================
1. このフォルダ内の "Utawave.app" を「アプリケーション」フォルダにドラッグしてください。

2. 初回起動時のみ（重要）:
   - Finder で Utawave.app を「右クリック → 開く」
   - 警告ダイアログで「開く」をクリック
   ※ 通常のダブルクリックだと macOS Gatekeeper にブロックされます
     （Apple の署名・公証を取得していない開発版アプリのため。中身は安全です）
   ※ 2 回目以降は通常のダブルクリックで起動できます

   ▼ もし「"Utawave" は壊れているため開けません」と出る場合（Apple Silicon）
     ターミナルで以下を 1 回だけ実行してください（アプリを移動後）:
       xattr -cr /Applications/Utawave.app
     これは zip ダウンロード時に付く検疫属性を外すだけのコマンドです。

===================================
■ 使い方
===================================
"MANUAL.html" をダブルクリックして Web ブラウザで開いてください。
インストール / 録音 / 編集 / MIDI / 書き出し / ショートカット一覧 を記載しています。

===================================
■ システム要件
===================================
- macOS 12 (Monterey) 以降
- Apple Silicon (M1 以降) 専用 ※ Intel Mac には未対応です

===================================
■ かんたんな試し方（5 分）
===================================
1. 起動 →「新規プロジェクト」
2. カラオケ音源を画面にドラッグ＆ドロップ（先頭 0 秒に並びます）
3. 録音したいトラックの「R」を点灯 → Space で再生しながら歌う → R で停止
4. 波形が出たら Space で聴き直し。気に入らなければ Cmd+Z で取り消し
5. メニュー → 書き出し（または Cmd+E）で WAV / MP3 に出力

===================================
■ 見ていただきたいポイント
===================================
- 録音 → 停止 → 波形がすぐ表示されるか、音ズレが無いか
- 書き出した WAV / MP3 が他のプレーヤーで正しく鳴るか
- マスターの音量メーター（赤＝0dB オーバー）と CLIP! / RESET の挙動
- 操作中にクラッシュ・フリーズ・文字化け・表示崩れが無いか
- （MIDI を使う方）MIDI トラック追加 → ピアノロールでの打ち込み

===================================
■ ファイルについて
===================================
- 録音やインポートしたオーディオは、プロジェクトフォルダ内の "Audio/" に保存されます
- 自動保存は "Backup/" に <プロジェクト名>.autosave.utawave として作成されます
- 書き出し既定先は "Export/" です
- 万一クラッシュしても、次回起動時に自動保存からの復旧を案内します
